  Emacs既是魔鬼也是天使，既高傲冷漠又谦逊卑微；既可让你深陷泥沼，也能让你健步如飞；在
某个时刻让你纵横驰骋，下一秒又让你堕入深渊......;Emacs是悖论的集大成者，交杂着拥踵者
无尽的爱与恨，一如那复杂难解的人生，写满成功与失败，记着冷雨和热风。
  不知觉之间已和它相处三载光阴矣！从艰难的快捷按键强记，到去抄袭大神们的配置，再去学习
common lisp和emacs lisp,再去编写自己的配置；期间不止十几次的想要放弃，真是受够了这
个魔鬼的折磨！还是卸了又装，改烂了从头再来，到如今终于算得上心手一体了；不管是coding还
是writing，不管是要做schedule还是想写paper，抑或是打游戏，浏览网页也并非不可以。emacs
本身就是一个lisp构建起来的操作系统，如深井中的甘泉，取之不尽，用之不竭。（you can forget
all the IDEs.)
  知识和行动啊！你们真的是很难合为一体啊。见多了渊博宽广的知识二道贩子，喋喋不休的给人
做宣贯，做指导；也看了太多的枭雄猛将，左突右支的冲锋陷阵，“颓然疲役而不知其所丧”。我们要
忍受这知行不一所带来的痛苦，困惑以及彷徨和孤独，真的要努力克服内心中想要获得即时满足感的
恶劣品性；在知识和行动中寻找一种动态的平衡。life is suffering, live is not that 
easy.但是却可以努力最求一种转换，就像在人生的求索中敲击这样的语句：
                    （reverse "evil") => "live"
与其在悲观和失望中消沉，以为人间恶多善少。还不如用生机去诅咒呆滞，用美好去诅咒丑恶，用
光明去诅咒黑暗。还有，自我转变，切莫急迫的想要改变他人！
